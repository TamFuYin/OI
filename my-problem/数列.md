# 数列

众所周知，出一道题的最好方式就是读错题。

## 题目描述

称一个数列是好的，如果对于任意正整数 $k$，$k$ 的倍数在数列中的出现都是连续的。

比如 $[1,2,6,3]$ 是好的，但是 $[1,2,3,6]$ 不是，因为 $2$ 的倍数的出现位置为 $2,4$。

对一个长度为 $m$ 的数列 $b$，记 $f(b)$ 为满足以下条件的 $b'$ 的数量：

- $b'$ 为长度为 $m$ 的正整数序列
- $b'$ 是好的
- 对于 $1\le i\le m$，$b_i|{b'}_i$ 且 $\gcd({b'}_i/b_i,b_i)=1$
- $\operatorname{lcm}_{i=1}^m\{b_i\}=\operatorname{lcm}_{i=1}^m\{{b'}_i\}$

现在 <a href="https://qoj.ac/user/profile/tanfuxuan" style="color:rgb(55, 205, 155)">tanfuxuan</a> 有一个长度为 $n$ 的数列 $a$，和 $q$ 次询问，每次询问给出 $l,r$ 你需要求出 $\sum_{x=l}^r\sum_{y=x+1}^rf(a[x:y])$，输出对 $998\ 244\ 353$ 取模的结果。

## 输入格式

第一行两个正整数 $n$ 和 $q$，保证 $1\le n,q\le 5\times 10^5$

接下来一行 $n$ 个正整数 $a_1,a_2,\ldots,a_n$，保证 $1\le a_i\le 1\times 10^9$

接下来 $q$ 行，每行两个正整数 $l$ 和 $r$，保证 $1\le l\le r\le n$。

## 输出格式

输出 $q$ 行，每行一个整数，表示对应的答案。

## 输入输出样例

#### 输入 #1

```
5 3
1 2 3 4 5
1 5
2 4
3 5
```

#### 输出 #1

```
?
?
?
```
