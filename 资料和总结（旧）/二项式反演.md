证明：$g(n)=\sum_{i=0}^n(-1)^i\binom n i f(i)\implies f(n)=\sum_{i=0}^n(-1)^i\binom n i g(i)$ 。
$$
\begin{aligned}
&\sum_{i=0}^n(-1)^i\binom n i g(i)\\
&=\sum_{i=0}^n(-1)^i\binom n i \sum_{j=0}^i(-1)^j\binom i j f(j)\\
&根据「三项式版恒等式」:\\
&=\sum_{i=0}^n(-1)^{i+j}\binom n j \sum_{j=0}^i\binom {n-j} {i-j} f(j)\\
&这时,我们改变求和顺序,并发现(-1)^{i+j}=(-1)^{i-j}\\
&=\sum_{j=0}^n\sum_{i=j}^n(-1)^{i-j}\binom n j \binom {n-j} {i-j} f(j)\\
&此时,令i\gets i-j,即用\ i+j\ 代替i\\
&=\sum_{j=0}^n\sum_{i=0}^{n-j}(-1)^i\binom n j \binom {n-j} i f(j)\\
&=\sum_{j=0}^n\binom n j f(j)\sum_{i=0}^{n-j}(-1)^i\binom {n-j} i\\
&事情已经很明了了,根据二项式定理:\\
&=\sum_{j=0}^n\binom n j f(j)\sum_{i=0}^{n-j}\binom {n-j} i1^{n-j-i}(-1)^i\\
&=\sum_{j=0}^n\binom n j f(j)(1-1)^{n-j}\\
&如果规定0^0=1,则有:\\
&=\sum_{j=0}^n\binom n j f(j)[n-j=0]\\
&=\sum_{j=0}^n\binom n j f(j)[j=n]\\
&=\binom n n f(n)\\
&=f(n)\\
&\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ □
\end{aligned}
$$
